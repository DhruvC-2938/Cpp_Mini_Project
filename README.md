## ğŸ§  OOP Concepts Explained
- **Class (`Hotel`)**  
  Represents the hotel and manages room details, pricing, availability, and billing.

- **Object (`h`)**  
  An instance of the `Hotel` class used to access booking and checkout operations.

- **Encapsulation**  
  Room data such as price, availability, and occupied rooms are kept private and accessed only through class methods.

- **Abstraction**  
  Users interact with the system using menu options without knowing internal data handling.
